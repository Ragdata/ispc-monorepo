# ISPC-Monorepo

The `ISPC-Monorepo` is one, HUGE, centralised collection of every package that you can install with `ISPC-Kit` and then maybe even a few little extras. This is a one-stop server commissioning suite which contains everything you need to create a world-class hosting appliance from 100% Open Source packages!
